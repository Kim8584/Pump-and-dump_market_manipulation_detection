The "Rush" Signature

Unlike organic trading which shows scattered latency, manipulation involves bots hitting the order book in the same 10ms window. This creates a vertical volume spike.

03. FEATURE ENGINEERING

01

Time Chunking

Continuous trade streams are bucketed into 15s snapshots. This allows the model to analyze 'pulses' of activity rather than individual data points.

02

Rolling Context

Every snapshot is compared against the previous 900 chunks (approx 3.75 hours). This normalization highlights anomalies relative to the daily average.

04. FEATURE DICTIONARY

The variables that drive the Random Forest Classifier's decision making.

std_rush_order

Standard deviation of inter-arrival times. Lower values = synchronized bot activity.

std_volume

Identifies massive surges in capital inflow that break the rolling baseline.

avg_price_max

Measures how aggressively the local resistance level is being breached.

hour_sin/cos

Cyclical time encoding to map 'pump likelihood' to specific UTC hours.
